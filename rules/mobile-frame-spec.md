# 手机端框架规范

> 绘制手机端原型（含小程序）时严格遵守。

## 容器规格

- 手机容器尺寸：`375px × 812px`，居中展示。
- 外层背景色：`#ebe7f3`。
- 不加圆角。阴影：`box-shadow: 0 12px 50px rgba(0, 0, 0, 0.25);`
- 内部滚动区域隐藏滚动条：

```css
scrollbar-width: none;
-ms-overflow-style: none;
&::-webkit-scrollbar { display: none; }
```

## Class 命名

| 元素 | 推荐 class |
|------|-----------|
| 手机容器 | `.phone` |
| 状态栏 | `.status-bar` |
| 小程序导航栏 | `.mini-program-bar` |
| 页面内容滚动区 | `.page-content` 或 `.page` |
| Home Indicator | `.home-indicator` |

### 弹窗/遮罩定位

- 所有弹窗、遮罩、Modal **必须**放在 `.phone` 容器内部。
- 使用 `position: absolute`（非 `fixed`），遮罩只覆盖 `.phone` 区域。
- `.phone` 设置 `position: relative` 作为定位基准。

## 微信小程序外壳

由四层组成（自上而下）：

1. **状态栏**（44px）：左侧时间（加粗）、中间灵动岛（`110×32px` 纯黑 `#000` 圆角药丸）、右侧信号+电量（不加 WiFi 图标）。
2. **导航栏**（44px）：居中标题、右侧胶囊按钮（`height:32px; border-radius:16px`，左"更多"三点+右"关闭"圆环，中间 `1px` 竖线分隔）。
3. **页面内容区**：承接具体业务内容。**所有业务内容必须在此区域内**。
4. **Home Indicator**（34px）：居中横条 `134×5px`，圆角 `2.5px`。

## 配色

外壳元素背景色跟随页面主题：

| 模式 | 外壳背景 | 文字/图标/横条 | 胶囊按钮背景 | 胶囊边框 |
|------|---------|-------------|------------|---------|
| 暗黑 | 深色（如 `#101218`） | 亮色（白/浅灰） | 略浅深色 | 半透明浅灰 |
| 亮色 | 浅色（如 `#ffffff`） | 暗色（黑/深灰） | 略深浅色 | 半透明深灰 |

灵动岛始终纯黑 `#000`，不受主题影响。外壳元素与页面背景需高对比度。
