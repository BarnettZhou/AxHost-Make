# 快捷键说明

以下快捷键在开发模式（Shell）和预览模式（Preview）中均可使用。当焦点位于输入框、文本域等可编辑元素内时，字母快捷键（I/T/D/N）不会触发，避免干扰文本输入。

---

## 开发模式（Shell）

| 快捷键 | 功能 |
|---|---|
| `I` | 抓取元素：开启/关闭 iframe 内的元素检查器，点击页面元素可获取其 HTML 片段和 CSS 选择器 |
| `T` | 触控模拟：开启/关闭移动端触控滚动模拟（在 iframe 内启用拖拽滚动） |
| `D` | 文档面板：显示/隐藏右侧文档面板 |
| `N` | 导航栏：显示/隐藏左侧目录树导航 |

---

## 预览模式（Preview）

| 快捷键 | 功能 |
|---|---|
| `T` | 触控模拟：开启/关闭移动端触控滚动模拟 |
| `D` | 文档面板：显示/隐藏右侧文档面板 |
| `N` | 导航栏：显示/隐藏左侧目录树导航 |

---

## 抓取元素模式（Inspector）

| 快捷键 | 功能 |
|---|---|
| `Esc` | 退出抓取：关闭元素检查器，恢复鼠标正常状态 |

---

## Prompt Box 自动补全

在 Prompt Box 中输入 `@` 或 `#` 唤起下拉框后，可使用以下快捷键：

| 快捷键 | 功能 |
|---|---|
| `↑` / `↓` | 上下切换选项 |
| `Enter` / `Tab` | 确认选中，替换为 `@pages/{hash}({名称})` 或 `#components/{hash}({名称})` |
| `Esc` | 关闭下拉框，保持当前输入不变 |
