# Prompt Box 功能说明

Prompt Box 位于开发模式右侧边栏底部，是用户与 AI 交互的入口。用户在输入框中输入修改需求，点击「复制提示词」后，系统会将当前页面上下文 + 用户需求组装成结构化 Prompt，写入剪贴板，供粘贴到 AI 对话中使用。

---

## 1. 提示词组装

点击「复制提示词」时，系统会自动组装包含以下信息的结构化提示词：

- **Current Page**：当前页面类型、项目信息、页面路径（相对/绝对）、文档/JS/CSS 路径
- **User Request**：用户输入的修改需求
- **Output Instruction**：指示 AI 根据需求修改源码或文档，并输出修改要点

**前置校验**：
- 未选择页面/组件时，提示「请先选择一个页面或组件」
- 未输入文本时，提示「请输入修改需求」

---

## 2. @ / # 快速引用（Mention）

在输入框中输入 `@` 或 `#` 可快速引用项目内的页面或组件，无需手动查找路径。

### 触发规则

| 触发符 | 搜索范围 | 替换结果示例 |
|---|---|---|
| `@` | 页面（pages） | `@pages/a1b2c3d4(首页)` |
| `#` | 组件（components） | `#components/b2c3d4e5(Menu)` |

**触发条件**：
- `@`/`#` 前面必须是行首或空白字符（空格、换行）
- `@`/`#` 到光标之间不能包含空白字符
- 兼容中文输入法输入

### 交互方式

1. **唤起**：输入 `@` 或 `#` 后，在光标下方弹出浮动下拉框
2. **搜索**：继续输入关键字，实时模糊匹配页面/组件名称或路径
3. **展示**：下拉框中显示 **名称** + **所在路径**（如 `首页 / 子页面`），方便识别同名项
4. **选择**：
   - 键盘 `↑` / `↓` 切换选项，`Enter` 或 `Tab` 确认
   - 按 `Esc` 关闭下拉框
   - 鼠标直接点击选中
5. **替换**：选择后自动将 `@关键字` 替换为 `@pages/{hash}({名称})` 格式

---

## 3. 状态提示

底部状态栏用于显示操作反馈，如：

- 复制成功 / 复制失败
- 前置校验未通过的提示

提示信息会在 3 秒后自动清空。
