# 项目管理

---

## 1. 新建项目

在项目首页（Home）点击 **「新建原型项目」**，在弹出的对话框中输入项目名称，点击 **「创建」** 即可。

**创建后自动打开**：新项目创建成功后，会自动在当前窗口打开该项目的开发模式（Shell），无需手动进入。

**项目命名建议**：使用简洁的中文或英文名称，便于在首页识别和管理。

---

## 2. 查看项目属性

在开发模式（Shell）页面，点击顶部工具栏的 **设置图标（齿轮）**，打开「项目设置」弹窗，可查看和修改以下信息：

| 属性 | 说明 |
|---|---|
| **项目名称** | 当前项目的显示名称，修改后即时生效 |
| **项目 ID** | 系统自动生成的 8 位标识，只读，不可修改 |
| **托管项目** | 与远程 AxHost 平台的关联项目（如未使用可忽略） |

**托管项目**：
- 若已关联远程项目，显示项目名称和「解除关联」按钮
- 若未关联，可通过搜索框查找并选择已有项目，或点击「+」直接创建新的远程项目并自动关联

---

## 3. 重命名项目

在项目设置的「项目名称」输入框中修改名称，点击 **「保存」** 即可。

**生效范围**：
- 开发模式顶部标题栏会立即更新
- 预览模式页面标题会在下次刷新后生效
- 项目首页（Home）的卡片名称会在下次加载首页时同步
