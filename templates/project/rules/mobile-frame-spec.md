# mobile-frame-spec

> 手机端原型页面的框架规范，如果当前正在绘制手机端原型，请严格遵守

## 容器规格

- 手机容器尺寸：`375px × 812px`
- 外层页面（浏览器窗口）背景色：`#ebe7f3`
- 手机容器居中展示于浏览器视口

## 容器外观

- 手机容器**不加圆角**
- 手机容器需添加一层阴影：`box-shadow: 0 12px 50px rgba(0, 0, 0, 0.25);`

## 滚动条

- 手机容器内部滚动区域**禁止显示滚动条**
- 需使用以下 CSS 隐藏滚动条：

```css
scrollbar-width: none;
-ms-overflow-style: none;

&::-webkit-scrollbar {
  display: none;
}
```

## Class 命名规范

手机端页面的 DOM 结构应保持以下 class 命名一致性：

| 元素 | 推荐 class |
|------|-----------|
| 手机容器 | `.phone` |
| 状态栏 | `.status-bar` |
| 小程序导航栏 | `.mini-program-bar` |
| 页面内容滚动区 | `.page-content` 或 `.page` |
| Home Indicator（手势横条区） | `.home-indicator` |

### 弹窗 / 遮罩定位规则

- 所有弹窗、遮罩、Modal 等覆盖层元素**必须放置在 `.phone` 容器内部**
- 遮罩层使用 `position: absolute`，而非 `position: fixed`，确保遮罩只覆盖 `.phone` 区域，不会溢出到整个浏览器视口
- `.phone` 容器本身需设置 `position: relative`，作为遮罩层的定位基准

## 微信小程序外壳（Mini Program Shell）

> 若需求为绘制小程序页面，需在 `.phone` 容器内叠加拟真外壳。

当前在核心代码（AxHost Make）中已提供通过框架直接生成小程序壳的功能，在实际内容编写时需要严格遵守如下规则。

### Shell 组成

小程序外壳由以下四层结构组成，自上而下依次排列：

1. **状态栏（Status Bar）**：高 `44px`
   - 左侧：时间文本（如 `9:41`），字体加粗
   - 中间：灵动岛（Dynamic Island），`110px × 32px`，纯黑圆角药丸，绝对居中
   - 右侧：信号图标 + 电量图标（**不加 WiFi 图标**）
     - 信号：4 条竖向矩形，由矮到高排列，宽度 `3px`，圆角 `1px`
     - 电量：外框容器 `22px × 11px`，带 `1px` 边框，右侧带 `2px` 宽极柱；内部填充区域宽度约 `70%`

2. **小程序导航栏（Mini Program Bar）**：高 `44px`
   - 标题居中展示
   - 右侧胶囊按钮（Capsule）：绝对定位
     - 外框：`height: 32px`，圆角 `16px`，带 `1px` 边框
     - 内部分为左右两区，中间以 `1px` 竖线分隔
     - 左侧「更多」：三个横向排列的小圆点，每个 `3px × 3px`，圆角 `50%`，点间距 `3px`
     - 右侧「关闭」：圆环套圆点，外环 `14px` 圆形带 `1.5px` 边框，内点 `4px` 实心圆点

3. **页面内容区**：承接具体业务内容

4. **Home Indicator（全面屏手势横条）**：区域高 `34px`
   - 居中展示一条横条：`134px × 5px`，圆角 `2.5px`

### 配色方案

状态栏、导航栏、Home Indicator 等外壳元素的背景色**必须跟随当前页面的主题背景色**，不可与页面内容产生割裂。

| 模式 | 页面背景 / 外壳背景 | 标题 / 图标 / 手势条 | 胶囊按钮背景 | 胶囊按钮边框 |
|------|---------------------|----------------------|-------------|-------------|
| **暗黑模式** | 深色系（如 `#101218`） | 亮色系（白 / 浅灰） | 比背景略浅的深色 | 半透明浅灰 |
| **亮色模式** | 浅色系（如 `#ffffff`） | 暗色系（黑 / 深灰） | 比背景略深的浅色 | 半透明深灰 |

通用原则：
- 外壳文字、图标、手势条的颜色应与页面背景形成高对比度
- 胶囊按钮背景与边框应保持与页面主题协调的深浅层次，避免突兀
- 灵动岛始终使用纯黑（`#000`），不受主题切换影响
